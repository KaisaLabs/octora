# Octora

Octora is a standalone private-feeling execution layer for Meteora LP actions on Solana.

## Current Scope

- Standalone web app
- SOL-first add liquidity
- Claim
- Withdraw and close
- Standard and Fast Private modes
- Internal privacy adapter seam

## Not In MVP

- Perfect anonymity
- Multi-asset routing
- Shielded wallet UX
- Generalized Solana trading privacy
